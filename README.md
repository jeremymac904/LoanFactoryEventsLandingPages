# Loan Factory Events Landing Pages

Multi-language, high-converting landing page for the **Loan Factory California Mortgage Expo VIP Afterparty** (May 7, 2026).

## Features

- **4 Language Support**: English, Spanish, Vietnamese, Mandarin Chinese
- **Custom RSVP Form**: Submits directly to Google Sheets (no Google Forms embed)
- **SEO/AEO/GEO Optimized**: JSON-LD structured data, geo meta tags, keyword optimization
- **HeyGen Video Placeholders**: Ready for AI avatar video embeds
- **Mobile-First Responsive**: WCAG 2.1 AA compliant
- **Netlify Deploy Ready**: One-click deploy from GitHub

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Google Apps Script (form backend)
- Netlify (hosting)

## Quick Start

```bash
npm install
npm run dev
```

## Deployment

1. Push to GitHub
2. Connect repo in Netlify
3. Deploy the Google Apps Script (see `google-apps-script.js`)
4. Add `VITE_FORM_ENDPOINT` environment variable in Netlify
5. Trigger a redeploy

## Form Setup

See `google-apps-script.js` for the Apps Script code that handles form submissions to your Google Sheet.
